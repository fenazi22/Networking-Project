# Networking Project

This repository is part of my networking and infrastructure learning track.

## Focus Area

- IP addressing
- Subnetting
- Network classes
- Basic network design
- Device grouping
- Host planning

## Project Goal

Design three subnetworks where each network represents a different class range: Class A, Class B, and Class C.

Each subnetwork should include at least three devices and support a maximum of eight hosts.

## Learning Outcomes

- Understand the difference between network classes.
- Practice subnet planning.
- Calculate usable host ranges.
- Organize devices inside subnetworks.
- Document network structure clearly.

## Related Skills

- Networking fundamentals
- Subnetting
- Infrastructure basics
- Linux and command line practice
